# Python Piscine — 42 Mastery Curriculum

A five-day intensive Python bootcamp covering fundamentals through object-oriented design, functional programming, data science, and microservices.

---

## py-0 — Python Basics ✅

Introduction to the Python type system and standard data structures. Exercises cover manipulating `list`, `tuple`, `set`, and `dict`; formatting time and dates; type inspection with `match/case`; handling `None` and NaN; parsing CLI arguments; counting character classes in a string; and reimplementing the built-in `filter` function from scratch.

**Key concepts:** built-in types, f-strings, `sys.argv`, `match/case`, type introspection.

---

## py-1 — Object-Oriented Programming ⬜

First contact with Python classes. Exercises focus on defining classes with `__init__`, `__str__`, and `__repr__`; using properties and setters; overloading operators; and building basic class hierarchies through inheritance.

**Key concepts:** classes, dunder methods, encapsulation, operator overloading, inheritance.

---

## py-2 — Advanced OOP & Functional Tools ⬜

Deeper dive into Python's object model and functional features. Covers decorators, abstract base classes (`ABC`), metaclasses, iterators and generators, and higher-order functions (`map`, `filter`, `reduce`).

**Key concepts:** decorators, `ABC`, generators, `__iter__`/`__next__`, closures.

---

## py-3 — Data Science Foundations ⬜

Introduction to the scientific Python stack. Exercises work with `numpy` arrays and vectorised operations, statistical aggregations, and data visualisation with `matplotlib`. Covers loading and processing datasets with `pandas`.

**Key concepts:** numpy, pandas, matplotlib, vectorisation, descriptive statistics.

---

## py-4 — Microservices ⬜

Building lightweight HTTP services in Python. Exercises introduce `Flask` (or `aiohttp`) to expose REST endpoints, handle routing, and exchange JSON payloads. The module ends with a small multi-service architecture where independent services communicate over HTTP.

**Key concepts:** Flask, REST, JSON, request/response cycle, inter-service communication.
